Validating user data
====================

Autoinstall user data is validated against a :doc:`JSON schema <../reference/autoinstall-schema>`
during runtime before it is applied. It can also be validated ahead of time by
using the `validate-autoinstall-user-data script <https://github.com/canonical/subiquity/blob/main/scripts/validate-autoinstall-user-data.py>`_
